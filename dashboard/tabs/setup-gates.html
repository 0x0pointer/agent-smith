Manual Setup Gates

Manual / physical prerequisites a skill declares (a hooked device, an emulator on the network, a UART / JTAG hookup). Non-blocking — the scan never waits on these. Choose Now / Defer / Skip, follow the runbook, then Verify setup to run the readiness probe. A passing probe confirms the setup is genuinely live and resumes any deferred work.

No manual-setup gates. Skills that ship a capabilities.yaml open them here when invoked.
